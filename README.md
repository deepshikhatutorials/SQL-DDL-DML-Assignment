# SQL DDL & DML Assignment

This repository contains the SQL assignment based on **DDL (Data Definition Language)** and  
**DML (Data Manipulation Language)** concepts.

The assignment demonstrates database creation, table operations, and data manipulation using SQL.

---

##  Assignment Details

**Database Name:** CollegeDB  
**Tool Used:** Online SQL Compiler (MySQL)  
**Language:** SQL  

---

## Files Included

- `college_assignment.sql`  
  Contains all SQL queries from Question 3 to Question 18, including:
  - Table creation
  - Alter table operations
  - Insert, update, delete, and select queries

- `SQL_Assignment_Full_Output.txt`  
  Contains output of all SELECT queries after executing the SQL commands.

- `README.md`  
  Description of the assignment and repository.

---

## Topics Covered

### DDL (Data Definition Language)
- CREATE TABLE  
- ALTER TABLE  
- DROP TABLE  

### DML (Data Manipulation Language)
- INSERT  
- SELECT  
- UPDATE  
- DELETE  

---

## How to Run

1. Open any **MySQL-compatible online SQL compiler**
2. Copy the code from `college_assignment.sql`
3. Paste and execute the queries step by step
4. Verify outputs using SELECT statements

## Purpose

This repository is created for **academic submission** and practice of SQL DDL and DML commands.
